# Library Management System

A menu-driven Library Management System developed in C using linked lists and file handling. This project helps manage library books and issue records efficiently while demonstrating core data structure concepts and modular programming techniques.

## Features

### Book Management
- Add new books
- Update book details
- Delete books
- Search books by:
  - Book ID
  - Book Name
  - Author Name
- View all available books

### Issue Management
- Issue books to users
- Return issued books
- Track issue and due dates
- View all issued books

### Data Persistence
- Save book records to files
- Load book records from files
- Save issue records to files
- Load issue records from files

## Data Structures Used

### Singly Linked List
The project uses singly linked lists to store:
- Book records
- Issue records

This allows:
- Dynamic memory allocation
- Efficient insertion and deletion
- Better memory utilization

## Technologies Used

- C Programming
- Structures
- Singly Linked Lists
- File Handling
- Dynamic Memory Allocation
- Modular Programming
